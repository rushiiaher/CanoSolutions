## üî† **Typography System**

**Primary Fonts**

* **Headlines:** `Inter` ‚Äì Clean, modern, trusted
* **Body:** `Poppins` ‚Äì Friendly, readable, geometrically balanced
* **Tech/Code Sections:** `JetBrains Mono` ‚Äì Optional for code snippets or developer-facing content

**Type Scale** (Based on 8px rhythm)

| **Element**     | **Size** | **Line-height** | **Weight** |
| --------------- | -------- | --------------- | ---------- |
| H1              | 48px     | 56px            | 700        |
| H2              | 40px     | 48px            | 600        |
| H3              | 32px     | 40px            | 600        |
| H4              | 24px     | 32px            | 500        |
| Body Large      | 18px     | 28px            | 400        |
| Body Regular    | 16px     | 24px            | 400        |
| Caption / Notes | 14px     | 20px            | 400        |

**Typography Guidance**

* Headlines: Slight negative letter-spacing (`-0.02em`) for sharp appearance
* Paragraphs: Max width \~65ch for legibility
* Use strong **white space** around typography to allow breathing room
